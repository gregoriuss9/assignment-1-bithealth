# assignment-1-bithealth

This repository contains `HTML` and `CSS` file for form section website. `Boostrap` CSS Framework was used in this work. 

In general, the web structure consist of these sections: 
```
├── nav-bar
│   ├── HOME
│   └── PRODUCT
│   └── GALLERY
│   └── BLOG
│   └── INVENTORY
├── status
│   ├── PHOTO
|   └── IDENTITY
│   └── STATUS
```
